2.4.2 CSS

  1. Inline, embedded and external styles:
    • Inline: At the Headings level of the Home Page (index.html)
    • Embedded: At the top of the Gallery below the title tag is the embedded css style.
    • External: The main.css, paragraph.css, button.css , style.css file in assets/css is my external css file.
  2. External style sheet = main.css, formats the appearance of paragraphs, lists and headers
  3. External style sheet = main.css, changes hypertext in their four state conditions.
  4. External style sheet = main.css, has classes and ids that defines document context.
  5. External style sheet = main.css, uses float attributes to control positioning on the page.
